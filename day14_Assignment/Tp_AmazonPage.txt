Test Plan Identifier: tp-amz-001

1. Introduction
This test plan describes the approach, scope, objectives, and responsibilities for testing the Amazon Registration Process.
The purpose is to ensure that new users can successfully create an account with valid details, receive proper error messages for invalid inputs, complete OTP verification, and log in without issues.

2. Test Items
The following items in the registration process will be tested:
Input fields: Name, Email/Mobile, Password, Re-enter Password
Continue and Create Account buttons
OTP Generation and Verification (Email/Mobile)
Error messages for invalid/blank inputs
Account creation with valid details

3. Features to Be Tested
Input field validation (Name, Email/Mobile, Password, Re-enter Password)
Error messages for invalid/blank inputs
OTP generation and handling: valid OTP, invalid OTP, expired OTP
Successful account creation with valid details
Post-registration login validation

4. Features Not to Be Tested
Conditions of Use link
Privacy Notice link

5. Approach
Testing Type: Manual Testing, Black Box Testing (Functional Testing)
